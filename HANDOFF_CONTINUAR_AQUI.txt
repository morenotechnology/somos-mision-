## CONTEXTO DEL PROYECTO

Nombre: Somos Misión Colombia
Tipo: SaaS Premium — Red de Embajadores Digitales
Cliente: Misiones Nacionales de la IPUC (Iglesia Pentecostal Unida de Colombia)
Propósito: Plataforma de gestión, distribución de contenido y gamificación
           para una red de ~5.000 embajadores digitales.

## UBICACIÓN Y COMANDOS

Comandos clave:

  npm run build    → Build de producción (pasa OK, cero errores)
  npm run preview  → Previsualizar build de producción

## STACK TECNOLÓGICO

- React 19 + Vite 8
- Tailwind CSS v4 (via @tailwindcss/vite — NO tailwind.config.js)
- Framer Motion (animaciones)
- React Router DOM v7 (routing)
- Zustand + zustand/persist (estado global con persistencia en localStorage)
- Recharts (gráficos: AreaChart, BarChart, PieChart)
- Lucide React (TODOS los iconos — cero emojis en UI)
- React Hot Toast (notificaciones tipo toast)
- Date-fns (fechas — importado pero disponible)

## ZUSTAND STORE — ACCIONES DISPONIBLES

  login(role)              → loguea como 'admin' | 'pastor' | 'multiplicador'
  logout()                 → desloguea y limpia sharing/missions
  addXP(amount)            → suma XP al usuario actual + muestra notificación flotante
  toggleSidebar()          → colapsa/expande sidebar desktop
  setMobileSidebar(bool)   → abre/cierra drawer móvil
  shareContent(contentId)  → registra compartido + suma 50 XP (idempotente)
  completeMission(id, xp)  → completa misión + suma XP (idempotente)
  toggleNotifications()    → abre/cierra panel de notificaciones

## NOTAS TÉCNICAS IMPORTANTES

- Tailwind v4: NO existe tailwind.config.js. La config va en index.css via @theme o directamente.
  Los colores custom se usan como clases arbitrarias: bg-[#1A237E], text-[#D4AF37]
  
- Para añadir una nueva página:
  1. Crear src/pages/NuevaPagina.jsx
  2. Agregar lazy import en App.jsx
  3. Agregar <Route> en App.jsx
  4. Si es protegida: dentro del ProtectedRoute
  5. Agregar al Sidebar si necesita nav item

- Para añadir un nuevo campo al store Zustand:
  1. Añadir el state y action en src/store/useAppStore.js
  2. Si debe persistir, añadirlo a partialize()

- Para usar skeleton loaders:
  import { ContentCardSkeleton, StatCardSkeleton } from '../components/common/LoadingSkeleton';
  Mostrar condicionalmente mientras loading === true

- Para usar Modal:
  import Modal from '../components/common/Modal';
  const [open, setOpen] = useState(false);
  <Modal open={open} onClose={() => setOpen(false)} title="Título" size="md">
    ...contenido...
  </Modal>
